# instagramd

Command line app to download images,videos from instagram by url

## Install instructions

#### Download release 
https://github.com/ernesto27/instagramd/releases

#### Make file executable
```sh
$ chmod +x instagramd
```
#### Download Image
```sh
$ ./instagramd https://www.instagram.com/p/CMuYUHdDUw4/
```

#### Download Video
```sh
$ ./instagramd https://www.instagram.com/p/CMuYUHdDUw4/
```
